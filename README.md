# 🕌 بوت إذاعة القرآن الكريم

بث مباشر 24/7 لإذاعات القرآن الكريم عبر GitHub Actions مجاناً

## ⚙️ الإعداد

### 1. Secrets المطلوبة في GitHub
اذهب إلى: Settings → Secrets → Actions → New repository secret

| الاسم | القيمة | مثال |
|-------|--------|------|
| `BOT_TOKEN` | توكن البوت من @BotFather | `123456:ABC...` |
| `RTMP_URL` | رابط RTMP بدون المفتاح | `rtmp://a.rtmp.youtube.com/live2` |
| `STREAM_KEY` | مفتاح البث | `xxxx-xxxx-xxxx` |
| `ADMIN_IDS` | ID بتاعك على تيليجرام | `123456789` |

### 2. تفعيل الـ Workflow
- اضغط Actions → Quran Radio 24/7 → Run workflow
